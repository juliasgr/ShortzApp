This project is an academic project developed during the third semester of the Software Engineering course. It was created as part of the Server-Side Programming class, where the professor guided the students in building a web application similar to TikTok and YouTube Shorts.

The main goal of the project is to learn and apply server-side development concepts while building a short-video platform.

In addition, this project will also be used in the Software Testing class, where the application's functionalities will be tested and evaluated as part of the course activities.
